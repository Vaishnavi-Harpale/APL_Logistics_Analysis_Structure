# 🚚 Delivery Performance, Delay Risk, and Logistics Efficiency Analysis in Global Supply Chain Operations

## 📌 Project Overview

This project analyzes delivery performance and logistics efficiency using real-world supply chain data from **APL Logistics (KWE Group)**. The objective is to identify delivery delays, evaluate shipping performance, monitor SLA compliance, and provide actionable insights through an interactive Streamlit dashboard.

The dashboard enables logistics teams to analyze shipment delays, compare shipping modes, identify regional bottlenecks, and monitor key logistics performance indicators (KPIs).

---

# 🎯 Problem Statement

Large-scale logistics operations often experience:

- Delivery delays resulting in SLA violations.
- Increased operational costs due to longer shipping times.
- Inefficient shipping mode selection.
- Poor visibility into regional delivery performance.
- Limited understanding of late delivery risk.

This project provides a data-driven analytics solution to improve logistics decision-making.

---

# 🎯 Objectives

- Perform comprehensive data cleaning and preprocessing.
- Calculate delivery delay gaps.
- Classify deliveries as Early, On-Time, or Delayed.
- Analyze delivery performance across regions and markets.
- Evaluate shipping mode efficiency.
- Identify high-risk delivery locations.
- Analyze customer segment impact.
- Build a professional Streamlit dashboard for interactive analytics.

# 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/APL_Logistics_Analysis.git
```

Move into the project directory:

```bash
cd APL_Logistics_Analysis
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

# ▶️ Run the Dashboard

Start the Streamlit application:

```bash
streamlit run app.py
```

The dashboard will open in your default web browser.
